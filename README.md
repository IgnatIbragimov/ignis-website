# IGNIS — сайт бренда

Статический сайт для fashion-бренда IGNIS: главная, каталог, страница товара, корзина и оформление предзаказа. Чистый HTML/CSS/JS, без фреймворков и сборщиков.

## Как открыть локально

> Важно: открывать через **локальный сервер**, а не двойным кликом по файлу.
> Каталог и товары загружаются из `products.json` через `fetch()`, который в браузере
> не работает с `file://` — нужен http-сервер.

```bash
cd "IGNIS website"
python3 -m http.server 8000
```
Затем откройте: **http://localhost:8000**

    checkout.js       — сводка заказа и отправка

## Технологии

- HTML5, CSS3 (custom properties, Grid, Flexbox), vanilla JS (ES6+)
- Шрифт: **TT Firs Neue Trial** — DemiBold (заголовки), ExtraLight (текст), подключён локально через `@font-face`
- Корзина хранится в `localStorage` (ключ `ignis_cart`)
- Адаптив: 375px / 768px / 1280px

## Как добавить или отредактировать товар

Все товары лежат в одном файле — **`products.json`**. Отдельный HTML на каждый товар
создавать НЕ нужно: `product.html` подставляет данные по `id` из URL
(`product.html?id=shirt-001`), а каталог рисует карточки автоматически.

### Добавить новый товар

1. Положите фото товара в `assets/images/`.
2. Откройте `products.json` и добавьте новый объект в массив `products`:

```json
{
  "id": "shirt-002",
  "name": "НАЗВАНИЕ ТОВАРА",
  "price": 6000,
  "currency": "RUB",
  "status": "available",
  "drop_eta": "2 недели",
  "description": "Короткое описание про крой и силуэт.",
  "fit_note": "Для какого силуэта подходит вещь.",
  "measurements": {
    "back_length_cm": 59,
    "width_cm": 51,
    "sleeve_length_cm": 65
  },
  "sizes": ["S", "M", "L"],
  "composition": "95% хлопок, 5% эластан",
  "images": [
    { "src": "assets/images/НОВОЕ-ФОТО-1.jpg", "alt": "Вид спереди" },
    { "src": "assets/images/НОВОЕ-ФОТО-2.jpg", "alt": "Вид сбоку" }
  ]
}
```

Поля:
- **id** — уникальный, латиницей без пробелов (используется в ссылке).
- **status** — управляет логикой дропа (см. ниже).
- **drop_eta** — срок отправки, показывается у кнопки предзаказа.
- **sizes** — список доступных размеров (кнопки на странице товара).
- **images** — первый кадр становится обложкой в каталоге; остальные — галерея.
  Если ракурсов меньше — просто укажите столько, сколько есть.

### Статусы дропа (`status`)

| Значение      | Поведение на странице товара |
|---------------|------------------------------|
| `available`   | Кнопка активна, «Предзаказ. Отправка через {drop_eta}» |
| `closed`      | Кнопки нет, «Дроп завершён» + поле email «уведомить о следующем дропе» |
| `upcoming`    | Тизер «Скоро», заказ недоступен |
